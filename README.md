# Raylib C Starter Project

## Prerequisites
- [raylib](https://www.raylib.com/) installed on your system
- `gcc` or compatible C compiler
- `make`
- `pkg-config` (for finding raylib flags)

## Build

```
make
```

## Run

```
./main
```

## Notes
- On Windows, you may need to adjust the Makefile or use a MinGW/MSYS2 environment.
- Ensure raylib is installed and accessible via `pkg-config`.
